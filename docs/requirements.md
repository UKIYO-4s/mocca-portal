# Moccaポータル 要件定義書

## システム概要
お食事処もっかと宿ばんしろうの業務管理ポータルシステム
スマホでの操作を主に想定

## 技術スタック
- Laravel + MySQL
- Inertia.js + React（TypeScript）
- Tailwind CSS
- Googleカレンダー連携

---

## 機能一覧

### 1. ばんしろう予約管理
電話予約時にスタッフが統一フォーマットで入力 → Googleカレンダーに自動同期

**入力項目：**
- お名前（必須）
- お名前カナ（必須）
- メールアドレス（必須）
- 電話番号（必須）
- 住所（必須）
- チェックイン日（必須）※15時から
- チェックアウト日（必須）※10時まで
- 宿泊人数（必須）※大人・子供の内訳
- お食事の有無（朝食・夕食あり / 席のみ予約 / 食事なし）
- 送迎の有無（必要 / 不要）※加計呂麻島瀬相港〜西阿室間
- オプションサービス
  - 特製弁当 ¥820（税込）
  - 特製オードブル ¥4,950（税込）
  - BBQセット ¥7,700（税込）
  - 加計呂麻島観光案内（半日）¥13,200（税込）
  - 塩作り体験（午前¥5,000 / 午後¥4,000）
  - マリングッズレンタル
- お支払い方法（現金 / クレジット / 事前振込）
- ご質問・ご要望
- 対応スタッフ名（必須）

---

### 2. もっか食事予約管理
宿泊予約と同時に、または単独で食事予約を受付

**入力項目：**
- 予約種別（朝食 / ランチ / ディナー）
- お名前
- 人数
- 到着予定時間
- 連絡先
- 先出メニュー（あれば）
- 対応スタッフ名

---

### 3. ランチ準備チェックリスト
- 管理者がチェック項目を設定・変更可能
- スタッフが各項目をチェック
- 対応スタッフ名を記録
- 完了時刻を記録

---

### 4. ディナー準備チェックリスト
- 管理者がチェック項目を設定・変更可能
- スタッフが各項目をチェック
- 対応スタッフ名を記録
- 完了時刻を記録

---

### 5. 備品管理
- 備品の在庫数管理
- 発注アラート
- 入出庫履歴
- 対応スタッフ名を記録

---

### 6. 清掃チェックリスト
- 場所別チェックリスト
- 管理者がチェック項目を設定・変更可能
- スタッフが各項目をチェック
- 対応スタッフ名を記録
- 完了時刻を記録

---

### 7. タイムカード機能
- 出勤・退勤打刻
- 勤務時間自動計算
- 月次レポート

---

## 権限管理

| 権限 | 説明 |
|------|------|
| アドミン | 全機能へのアクセス、システム設定 |
| 管理者 | チェックリスト項目の変更、一部権限の譲渡、レポート閲覧 |
| スタッフ | 予約入力、チェックリスト入力、タイムカード打刻 |

## Googleカレンダー連携
- ばんしろう予約 → 指定カレンダーに自動追加
- もっか食事予約 → 指定カレンダーに自動追加
- 統一フォーマットでイベント作成
